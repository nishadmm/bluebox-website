There are no SAFETY preventions taken by small fishing Boats.

So we planned to raise a project for their safety that could help in any such situation, One such problem that fishermans faces when they go to sea.If the boat sinks or under any circumstances the boat is still in the middle sea,the officers can't be found the correct locations. BlueBox Innovations proposes a boat monitoring system using gsm gps and gyrosensor.The function sends the alert Message with location of accident to the coastal police station using only a single click while any obsatcles during sailing. If they can't send message while sinking, the functions automatically send that alert message to nearby coastal police station.The police can identify the location of accident in the form of google map link, derived from the latitude and longitude from Gps module.
